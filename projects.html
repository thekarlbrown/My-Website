This tool was an experiment in sending SMS messages via SMTP en-masse to cellular devices, without having to input SMTP server details or messages at runtime or know the carrier beforehand. It is a good mass SMTP request stresser for your SMTP server, and can be quite a chuckle if consent is obtained. Do not violate any local/state/federal laws while using this script.
